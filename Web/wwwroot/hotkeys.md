##При закрытой панели "Свойства"
| Комбинация клавиш              | Действие                                         |
|--------------------------------|:-------------------------------------------------|
| F1                             | Показ помощи (горячие кливиши + Интерфейсы)      |
| CTRL + n                       | Новый документ                                   |
| CTRL + o                       | Открыть документ                                 |
| CTRL + s                       | Сохранение документа                             |
| CTRL + Shift + s               | Сохранение документа под новым именем            |
| CTRL + z                       | Отменить последнее изменение                     |
| CTRL + Shift + z               | Применить последнее изменение                    |
| CTRL + e                       | Новая система                                    |
| CTRL + f                       | Поиск по имени интерфейса / системы              |
| Del                            | Удаление выбранного элемента                     |
| Shift + Del                    | Удаление группы выбранных элементов              |
| Esc                            | Закрыть текущее модальное окно                   |
| Up / Down /Left /Right         | Перемещение области видимости на 1 пиксель       |
| Shift + Up / Down /Left /Right | Перемещение области видимости на 10 пикселей     |
| Ctrl + Up / Down /Left /Right  | Перемещение области видимости на 100 пикселей    |
| Ctrl + l                       | Последовательное соединение выделенных элементов |
| Ctrl + m                       | Автоматическое выравнивание выделенных элеметнов |


##При открытой панели "Свойства" для системы

|--------------------|------------------------|
| CTRL + f           | Новая функция системы  |
| CTRL + d           | Новая сущность системы |
